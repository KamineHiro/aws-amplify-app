# Todoアプリケーション

シンプルで使いやすいTodoリスト管理アプリケーションです。

## デプロイ先

アプリケーションは以下のURLでご利用いただけます：
https://main.ddejv3a8xaumi.amplifyapp.com

## 主な機能

- ✨ タスクの追加
- ✅ タスクの完了/未完了の切り替え
- 🗑️ タスクの削除
- 💾 ブラウザでのデータ保持

## 使用技術

- React 19
- TypeScript
- Vite
- モダンなCSSスタイリング

## ローカル開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

その後、ブラウザで http://localhost:5173 を開いてアプリケーションにアクセスできます。

## ビルドコマンド

```bash
# プロダクションビルド
npm run build

# ビルドのプレビュー
npm run preview
```
